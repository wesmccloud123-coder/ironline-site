How to add an article:
1. Open articles.js
2. Copy an existing object
3. Change slug, title, summary, category, tags, and sections
4. Save
5. Commit and push

How to add a case study:
1. Open case-studies.js
2. Copy an existing object
3. Change slug, title, summary, tags, problem/change/outcome fields, and details
4. Save
5. Commit and push

URL examples:
- article.html?slug=slow-follow-up-cost
- case-study.html?slug=lead-follow-up-visibility
